## Installing

```shell
git clone https://github.com/derhuerst/moral-o-mat-bot.git
cd moral-o-mat-bot
npm i --production
```


## Usage

```shell
env TOKEN=my-telegram-bot-token node index.js
```

Get a token using [BotFather](https://telegram.me/BotFather).
